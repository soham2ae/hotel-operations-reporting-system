# Hotel Operations & Reporting System (HORS)

## Overview

HORS is a Hotel Operations & Reporting System designed to replace manual Excel-based hotel management workflows.

The system manages:

- Bookings
- Daily Room Entries
- Payments
- Pending Amount Analysis
- Room Transfers
- Early Checkout
- Additional Charges
- Hotel Reporting
- Power BI Dashboards
